# Neo4J - Análise de Dados com Grafos.

Este repositório reúne três projetos desenvolvidos como desafios de projeto do curso [Neo4j – Análise de Dados com Grafos](https://web.dio.me/track/neo4j-analise-de-dados-com-grafos), oferecido pela plataforma [DIO](https://web.dio.me/play).

Os projetos têm como objetivo aplicar, na prática, conceitos fundamentais de bancos de dados orientados a grafos, explorando modelagem, relacionamentos, consultas e análises de dados com foco em cenários reais. Cada desafio aborda um problema distinto, permitindo consolidar o uso do Neo4j como ferramenta para análise de dados complexos e altamente conectados. 

Durante o desenvolvimento das atividades, foi utilizada a ferramenta gratuita e online [Neo4j Aura](https://neo4j.com/books/definitive-guide-graph-databases-rdbms-developer/?utm_source=GSearch&utm_medium=PaidSearch&utm_campaign=Evergreen&utm_content=AMS-Search-SEMCO-Evergreen-None-SEM-SEM-NonABM&utm_term=neo4j+developer+guide&utm_adgroup=graph-database-guide&gad_source=1&gad_campaignid=20769287003&gbraid=0AAAAADk9OYoS0GoUOXYO_2B2ex7FviyfZ&gclid=CjwKCAiAncvMBhBEEiwA9GU_fpg_g-8BDatofAegxC_tCw2CpehGYyVRbRMwF-cCLZZtfzDtjKFL5RoCzn8QAvD_BwE), que permitiu a criação e execução das consultas em ambiente cloud, facilitando a implementação e os testes dos modelos propostos.

Durante o curso, foram desenvolvidos três projetos práticos, sendo eles:

- Sistema de Streaming com Neo4j

- Sistema de Recomendação Musical Baseado em Grafos com Neo4j

- Análise de Dados de Redes Sociais por meio de Consultas em Grafos

Os projetos desenvolvidos ao longo do curso Neo4j – Análise de Dados com Grafos, da DIO, permitiram aplicar na prática conceitos fundamentais de modelagem e análise de dados orientados a grafos utilizando o Neo4j.

As atividades proporcionaram uma visão mais aprofundada sobre como representar relações complexas, estruturar consultas eficientes em Cypher e extrair insights estratégicos a partir das conexões entre dados.

Além do aprendizado técnico, os desafios contribuíram para o desenvolvimento do raciocínio analítico e da capacidade de modelar problemas reais sob a perspectiva de grafos — uma abordagem cada vez mais relevante em cenários de recomendação, redes sociais e análise de comportamento.
